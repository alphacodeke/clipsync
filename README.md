A clean, modern web application that allows users to copy text to clipboard with one click. Features a beautiful UI, real-time status feedback, and fallback support for older browsers.

## ✨ Features

- 🚀 **One-click Copy** - Instantly copy any text to clipboard
- 🎨 **Modern UI** - Clean gradient design with smooth animations
- ✅ **Visual Feedback** - Success/error status messages with color coding
- 🔄 **Fallback Support** - Works on older browsers using `execCommand`
- ⌨️ **Keyboard Shortcut** - `Ctrl+Enter` (or `Cmd+Enter`) to copy
- 📱 **Responsive Design** - Works perfectly on mobile and desktop
- 🧪 **Sample Text** - Quick demo functionality
- 🧹 **Clear Button** - Easy input clearing

## 🛠️ Technologies Used

- **HTML5** - Semantic structure
- **CSS3** - Flexbox, gradients, animations, responsive design
- **JavaScript (ES6+)** - Async/await, Clipboard API, DOM manipulation
